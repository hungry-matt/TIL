# Commit message rule
~~~
- [#??] feat : 새로운 기능을 추가하였다.
- [#??] fix : 어떠한 버그를 고쳤다.
- [#??] style : 스타일을 바꾸었다. (CSS, HTML)
- [#??] refactor : 어떤 함수를 리팩토링 했다.
- [#??] test : 무언가를 테스트 했다.
- [#??] docs : 문서를 업데이트 했다.
- [#??] chore : 패키지 및 빌드 관련 프로젝트 설정을 했다.
~~~
